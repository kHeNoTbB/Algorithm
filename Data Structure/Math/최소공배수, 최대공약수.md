# 최대공약수 최소공배수

### 유클리드 호제법

```
public class Solution {
	public static void main(String[] args) {
		//최대공약수 gcd(big, small)
		System.out.println(gcd(12, 3));
		
		//최소공배수
		System.out.println((12*3)/gcd(12,3));
	}
	public static int gcd(int a, int b) {
		if(b==0) return a;
		return gcd(b, a%b);
	}
}
```
